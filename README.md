# Spectroscopic Models

This repository contains data and code for the manuscript on spectroscopic models.

## Contents

- `fft/`, `lt/`, `neon/`, `predict/`: data and analysis files
- `.org` files and other manuscript-related source material
- `.pkl` files for serialized data artifacts (ignored by `.gitignore`)

## Git setup

- `.gitignore` excludes common OS/IDE files and large binary pickle files (`*.pkl`).
- If you want to track `.pkl` files later, remove that line from `.gitignore`.
